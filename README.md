Talent Attraction Analyst - Technical Challenge
Overview

This repository contains my technical assessment for a Talent Attraction Analyst position.

The objective was to analyze recruitment data, identify patterns across the hiring funnel, and provide actionable recommendations to improve Talent Acquisition performance.

Deliverables
📄 Recruitment Data Analysis Report (PDF)
📊 Presentation (PowerPoint)

Methodology
The analysis included:
Exploratory Data Analysis (EDA)
Recruitment Funnel Analysis
Sourcing Channel Comparison
Conversion Rate Analysis
Candidate Profile Analysis
Business Insights
Data-driven Recommendations

Main Findings
GitHub, Inbound, and Hunting achieved the highest hiring conversion rates.
Candidate engagement proved to be the strongest predictor of hiring success.
Technical assessment performance strongly correlated with final hiring decisions.
Funnel optimization opportunities were identified in sourcing strategy and candidate qualification.
Tools Used
Microsoft Excel
ChatGPT (AI-assisted analysis and report structuring)
